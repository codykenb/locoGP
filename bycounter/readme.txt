1. Look at the documentation to find out what ByCounter can do
2. Request password to download ByCounter (please specify if you need the sources as well)
3. Execute and study the included examples, than use ByCounter for your own project

Contents of this page (http://sdqweb.ipd.kit.edu/eclipse/ByCounter/)
- /dist: password-protected area for downloading ByCounter (** denotes the increasing build number):
         ByCounterV0.9-b***.jar 		: executable ByCounter classes (without sources), requires the 9 JAR libraries from /lib (see below)
		 *Recommended*: ByCounterV0.9-b***_bin.zip		: executable ByCounter classes (without sources), INCLUDING the documentation and 9 JAR libraries from /lib (see below)
		 [on request, passwort-protected] ByCounterV0.9-b***_full.zip	: executable ByCounter classes (WITH sources), INCLUDING the documentation and 9 JAR libraries from /lib (see below)
		 [on request, passwort-protected] ByCounterV0.9-b***_src.zip		: ByCounter sources WITH required libraries (incl. documentation, no executable classes), 
		 [on request, passwort-protected] ByCounterV0.9-b***-src.jar		: ByCounter sources without required libraries (no documentation, no executable classes)
- /doc/api: Javadocs (a short introduction with examples can be found at the overview page, http://sdqweb.ipd.kit.edu/eclipse/ByCounter/doc/api/overview-summary.html )
- /doc/manual/ByCounter_Manual.pdf: Manual as a PDF file
- /lib: 9 libraries, required to executed ByCounter
- [on request, passwort-protected] /src: sources for browsing
